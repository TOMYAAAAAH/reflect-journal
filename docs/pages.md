# Reflect Journal - Front Pages

## Summary

|  | Name     | Description                             |
|--|----------|-----------------------------------------|
|  | Year     | Calendar with the whole year in a sight |
|  | Month    | Calendar month by month                 |
|  | Day      | Question and past answers               |
|  | Profile  | Parameters                              |
|  | Login    | Login                                   |
|  | Register | Register                                |

## Year
- [ ] Shows a grid of 3*4 months
- [ ] Clicking on a month navigates to the month page
- [ ] A day with an answer on the current year is highlighted
- [ ] The current day is highlighted
- [ ] Clicking on the "today" button navigates to the current month page
- [ ] Shows answer statistics

## Month
- [ ] Shows rows of 7 days
- [ ] Clicking on a day navigates to the day page
- [ ] Day with an answer is highlighted
- [ ] The current day is highlighted
- [ ] Clicking on the "today" button navigates to the current day page (or scrolls to the current month if not already
  on it)

## Day
- [ ] Shows the date
- [ ] Shows the question
- [ ] Shows past and current answers
- [ ] Allow to edit any answer

## Profile
- [ ] Shows user information :
    - name
    - email
    - creation date
- [ ] Allow user actions :
    - disconnect
    - erase account
    - empty account

## Login
- [ ] Allow user to log in with email and password
- [ ] Shows if email is not registered
- [ ] Shows if password is incorrect
- [ ] Offer to register

## Register
- [ ] Allow user to register with email and password
- [ ] Shows if email is already registered
- [ ] Shows two passwords fields to confirm
- [ ] Shows if passwords do not match
- [ ] Offer to log in
