# User Flows

## New user
- arrives on welcome page
- is prompted to log in
- registers
- has empty answers
- is prompted to write

## Logged in user, new phone
- arrives on welcome page
- is prompted to log in
- logs in
- retrieve their answers

## Logged in user, loging out
- goes to profile
- logs out
- is redirected to today
- is prompted to login

## Any user, server errors
- sees the day of today
- sees an excuse for not displaying the question
- no prompt to connect and no answers
